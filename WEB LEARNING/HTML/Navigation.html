Navigation in HTML

  • Anchor Tag ( <a> )
    • The anchor tag is used to create a hyperlink, allowing users to navigate from one webpage to another.
    • Attributes:
      • href: Specifies the destination URL.
      • target: Defines where the link will open.
        1. _self: Opens the link in the same tab (default).
        2. _blank: Opens the link in a new tab.
    • Examples:

      - Open in same tab:
      <a href="https://www.google.com">Visit Google</a>
      Result: Visit Google

      - Open in new tab:
      <a href="https://www.google.com" target="_blank">Visit Google in New Tab</a>
      Result: Visit Google in New Tab

  • Image Tag ( <img> )
    • Used to display images on a webpage.
    • Attributes:
      1. src: Source or path of the image file.
      2. alt: Alternative text if the image fails to load.
      3. width & height: Set image dimensions in pixels.
    • Example:

      <img src="https://images.pexels.com/photos/1323550/pexels-photo-1323550.jpeg?cs=srgb&dl=pexels-simon73-1323550.jpg&fm=jpg" alt="Sample Image" width="200" height="120">

  • Audio Tag ( <audio> )
    • Used to embed audio content (.mp3 files).
    • Attributes:
      • src: Path of the audio file.
      • controls: Displays audio playback options.
    • Example:

      <audio src="https://www.w3schools.com/html/horse.mp3" controls></audio>

      Result:

  • Video Tag ( <video> )
    • Used to display video files on a webpage (.mp4, .webm).
    • Attributes:
      1. src: Source or path of the video file.
      2. controls: Adds play/pause, volume, etc.
      3. autoplay: Automatically plays video on page load.
      4. muted: Mutes video by default.
      5. loop: Repeats video continuously.
      6. width & height: Set the video dimensions.
    • Example:

      <video src="https://www.w3schools.com/html/mov_bbb.mp4" width="300" height="200" controls autoplay muted loop></video>

      Result:

  • Iframe Tag ( <iframe> )
    • Used to embed another HTML page inside the current page.
    • Attributes:
      • src: URL of the external page.
      • width & height: Set the size of the iframe.
      • frameborder: Adds/removes a border around the iframe.
      • name: Used as a target for links.
    • Example:

      <iframe src="https://www.wikipedia.org" width="300" height="200" frameborder="1" name="myFrame"></iframe>

      Result:

    • Targeted Link Example:
      <a href="https://www.google.com" target="myFrame">Open Google in iframe</a>

      Result:
      Open Google in iframe
